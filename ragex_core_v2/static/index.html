About RAG-Ex Core

RAG-Ex Core is an explainable Retrieval-Augmented Generation (RAG) system that provides transparency into how language models generate answers based on retrieved context.

Features

  • Explainable AI: Understand which parts of the context are most important
  • Multiple Retrievers: Choose between dense semantic search, BM25 keyword matching, or hybrid
  • Confidence Threshold: Prevents hallucinations by refusing to answer when retrieved documents have low relevance (default threshold: 0.6)
  • Interactive Highlighting: Adjust importance thresholds with a slider to visualize context relevance
  • Perturbation Analysis: Various strategies to test context importance
  • Flexible Granularity: Explain at word, phrase, sentence, or paragraph level
  • Debug Mode: Deep dive into perturbation details and similarity scores
  • Multiple Comparators: Different methods to measure answer similarity

API Endpoints

POST /explain
Main endpoint for explainable RAG. Retrieves context (or uses provided context), generates answer, and provides importance scores for each text unit.

POST /retrieve
Retrieve relevant documents for a given query using your selected retrieval method.

GET /health
Check API health status.
